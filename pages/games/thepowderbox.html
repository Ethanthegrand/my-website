The Powder Box

Play on Roblox! (Primary version)
Play on Roblox! (All ages version)

The Powder Box is a fully fledged 2D falling sand game on Roblox. This game was my first attempt at a fully fledged non-3D game made with a custom 2D pixel rendering engine.

This game is the first large successful Roblox port of a falling sand game on the entire platform.



As a falling sand game, this game offers many features and content. As of version 1.22, there are over 100 elements and objects (including over 30 hidden elements), over 442,000 published creations, sound simulation, 2D box physics for entities, circuitry simulation, explosions, and much more!

This game has support for all devices (Desktop, Mobile, Tablet and Console)

This game was originally a small demo to test to capabilities the custom engine I made, but as the project grew larger into an actual fun sandbox game, I released the game to the public on the 29th of March 2022.

As of the 26th of September 2025, The Powder Box has over 82,700,000 total visits (from the two versions of the game combinied).
